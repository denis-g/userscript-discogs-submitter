# Security Policy

## Scope

This is a client-side userscript. No data is stored on external servers, and no user data is collected or transmitted
to any third-party infrastructure besides the official Discogs API and the specific music stores the script supports.

## Reporting a Vulnerability

If you suspect a vulnerability in the parsing logic or script handling, please report it via profile links at https://github.com/denis-g.

Please do not report potential security issues in public GitHub issues to avoid exposing them before a fix can be applied.
